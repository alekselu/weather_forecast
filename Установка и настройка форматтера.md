# Установка и настройка форматтера

Автоформатирование `python` файлов при коммите осуществляется через пакеты `black` и `pre-commit`. Для настройки запустите `setup_precommit.py`:

```bash
python3 setup_precommit.py
```

Далее автоформатирование будет доступно при выполнении коммита. В случае, если некоторые файлы были неверно отформатированы, необходимо осуществить коммит еще раз.
